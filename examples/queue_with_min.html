Description

queue_with_min public construct/copy/destruct

  1. queue_with_min() noexcept;

    Complexity: O(1)

    Note: fibonacci_heap() does not throw, but is not marked with noexcept. Need to define move contructor/assignments and default constructors, instead of writing = default

    https://github.com/boostorg/heap/issues/5 

  2. queue_with_min(queue_with_min && q) noexcept;

    Complexity: O(1)

    Note: After deque move assignment/construction (unless elementwise move assignment is forced by incompatible allocators) references, pointers, and iterators (other than the end iterator) to other remain valid, but refer to elements that are now in *this. The current standard makes this guarantee via the blanket statement in §23.2.1[container.requirements.general]/12, and a more direct guarantee is under consideration via LWG 2321
